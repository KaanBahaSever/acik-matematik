# math-notebook

Open-source, interactive mathematics lecture notes for undergraduate level in Turkish.

## Kurulum

```bash
npm install
```

## VitePress + MathJax Kurulum Komutları

```bash
npm init -y
npm install -D vitepress markdown-it-mathjax3 markdown-it-container
```

## Çalıştırma

```bash
npm run docs:dev
```

## Derleme

```bash
npm run docs:build
```
